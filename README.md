# 🍽️ Zicron Tasty Spot

Welcome to **Zicron Tasty Spot** — a restaurant website proudly designed and developed by high school students of **Zicron School**.

This project showcases our creativity, technical skills, and passion for building real-world web applications.

---

## 📖 About the Project

Zicron Tasty Spot is a simple and user-friendly restaurant website that allows visitors to:

- View our menu 🍔🍕
- Learn about the restaurant 🏢
- Explore featured dishes 🍽️
- Contact us 📞

The goal of this project was to create a functional and visually appealing website while learning web development fundamentals.
